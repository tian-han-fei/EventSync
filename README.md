# EventSync

EventSync is a website for local event planning and community-building.

## Description

EventSync features a minimalist layout and account system to post event details on a public board. Potential also exists for a friend/DM system to be implemented later on.

## Installation

File structure can be downloaded and repurposed for per-device use.

## Usage

Currently hosted on url at 000webhost.
